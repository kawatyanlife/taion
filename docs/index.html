簡単報告

提携文を使って報告
キーボードで文章を入力する必要も無し

らくらく登録

起動した瞬間にわかる直感的なUI
入力はキーボード不要で最大2タップで登録が可能

メールやLINEで送信

登録した体温は報告ボタンからメールやLINEで送信
Webサイトなどがある場合はそのURLを登録すれば、体温をコピーしてアクセスすることができる！

提携文

送信する本文は登録することができるので、「お疲れ様です。XXです」などの決まった文章は自動で入力されます。もちろん提携文の変更も可能です。

体温が色でわかる

理想的な体温は36.5～37.1度と言われています。理想的な体温の場合は青色、少し低かったり高かったりする場合は黄色、低すぎたり高すぎる場合は赤色で表示
目で体温の状況を確認できる！
